# getUserName

---

This is a minimalist library. You can use it to get current user name of system.

## Demo

```c++
  #include <iostream>
  #include "getUserName.h"

  using namespace std;

  int main(int argc, char* argv[])
  {
      cout << "Current user name is " << getUserName() << "." << endl;
      return 0;
  }
```
